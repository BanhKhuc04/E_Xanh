# Mẫu báo lỗi E-XANH

```text
Tên lỗi:
Mã testcase liên quan:
Trang/route bị lỗi:
Tài khoản đang dùng: Guest / User / Admin
Thiết bị: Desktop / Mobile / Tablet
Trình duyệt:
Các bước tái hiện:
1.
2.
3.

Kết quả hiện tại:
Kết quả mong muốn:
Ảnh/video minh chứng:
Mức độ lỗi: Low / Medium / High / Critical
Ghi chú thêm:
```

## Ví dụ lỗi nên báo ngay

- Bấm nút không có phản hồi.
- Route bị 404/trắng trang.
- User thường vào được admin.
- Đăng bài cộng đồng nhưng hiện ở mẹo tiết kiệm.
- Bài đã lưu bị mất sau reload.
- Nút hover/click bị trắng hoặc mất chữ.
- Mobile bị tràn ngang.
- Icon avatar/setting/chuông/dấu hỏi nhìn như bấm được nhưng không hoạt động.
